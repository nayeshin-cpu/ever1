## 2. 핵심 개념: "그 아이만의 모습"

### 왜 중요한가

AI가 만든 영상이 "그냥 골든리트리버"가 아니라 **"우리 콩이"** 로 느껴져야 한다.
같은 품종이어도 꼬리 흔드는 속도, 걷는 방식, 에너지가 다 다르다.
이 차이를 만드는 것이 EverPet의 핵심 가치다.

### 구현 방향 (Phase별)

```
Phase 1 (MVP) — 보호자가 직접 입력
  └→ 에너지 레벨 + 성격 키워드 선택
  └→ 선택값이 AI 프롬프트로 변환되어 영상에 반영

Phase 2 — 기존 영상 분석 (선택 기능)
  └→ 보호자가 기존 영상 업로드
  └→ AI가 행동 패턴 자동 분석
  └→ 더 정확한 개체 맞춤 생성

Phase 3 — 누적 데이터 기반 자동 학습
  └→ DBGD 데이터 축적
  └→ 품종별·개체별 행동 모델 고도화
```

## 3. 입력 데이터 스펙

### 3.1 필수 입력 — 사진

| 항목 | 스펙 |
|------|------|
| **지원 포맷** | **JPG, JPEG, PNG** |
| **최대 파일 크기** | **10MB** |
| **권장 해상도** | **2K (2048px) 이상** |
| 최소 해상도 | 512px 이상 (단변 기준) |
| 권장 구도 | 반려동물 전신 또는 상반신, 정면~45° 측면 |
| 피해야 할 구도 | 반려동물이 화면의 20% 미만, 뒷모습만, 극단적 측면 |
| 배경 | 상관없음 (AI가 자동 제거) |

**업로드 후 자동 처리:**
```
JPG/PNG 수신 (최대 10MB)
  → 파일 유효성 검사 (포맷, 크기, 해상도)
  → Cloudflare R2 원본 저장
  → 배경 자동 제거 (rembg)
  → 림라이트 추가 (홀로그램 최적화)
  → 깊이 추정 (Depth Anything v2)
  → AI 영상 생성 준비 완료
```

**업로드 품질 피드백 (실시간):**

| 해상도 | 사용자에게 표시 |
|--------|--------------|
| 2K (2048px) 이상 | "선명한 사진이에요 👍" |
| 1K~2K | "괜찮아요. 더 선명하면 더 좋아요" |
| 512~1K | "조금 더 선명한 사진을 올려주시면 더 좋아요" |
| 512 미만 | "사진이 너무 작아요. 더 선명한 사진을 올려주세요" (진행은 가능) |

### 3.2 필수 입력 — 반려동물 기본 정보

| 항목 | 입력 방식 | 필수 | AI 활용 방식 |
|------|---------|------|------------|
| 이름 | 텍스트 | ✅ | 화면 표시용 |
| 품종 | 검색 자동완성 (한글/영문) | ✅ | 품종별 기본 행동 패턴 적용 |
| 체형 | 소형 / 중형 / 대형 선택 | ✅ | 움직임 비율·속도 조정 |
| 털색 | 사진에서 자동 추출 or 팔레트 선택 | 권장 | 영상 색보정 |
| 나이 | 퍼피 / 성견 / 노견 선택 | 권장 | 에너지 레벨 기본값 |
| 성별 | 선택 | 선택 | 체형 비율 미세 조정 |

### 3.3 핵심 입력 — 행동 특성 ⭐ (개체 맞춤의 핵심)

보호자가 우리 아이를 가장 잘 안다. 보호자의 선택이 곧 개체 맞춤 영상의 재료다.
**처음 등록 시 1회 입력 → 이후 생성마다 자동 적용.**

---

**Step 1 — 에너지 레벨** (1개 선택, 필수)

```
○ 항상 신나있고 활발했어요    → energy: high
○ 보통이었어요               → energy: medium
○ 조용하고 차분했어요        → energy: low
```

---

**Step 2 — 성격 키워드** (최대 3개 선택, 선택 안 해도 됨)

```
□ 꼬리를 엄청 세게 흔들었어요       → tail_wag: strong + fast
□ 항상 졸졸 따라다녔어요            → follow_owner: true
□ 공·장난감을 물어오는 걸 좋아했어요 → play_fetch: true
□ 밥·간식 앞에서 특히 신났어요      → food_motivated: true
□ 낯선 사람한테도 먼저 달려갔어요   → social_stranger: high
□ 조용히 옆에 있는 걸 좋아했어요    → calm_presence: true
□ 산책 나가면 코를 바닥에 박았어요  → sniff_focused: true
□ 이름 부르면 바로 달려왔어요       → recall_response: strong
```

---

**Step 3 — 무드** (매 영상 생성마다 선택, 필수)

```
○ 포근한  — 쉬거나 옆에 있는 모습
○ 활발한  — 뛰거나 신나게 노는 모습
○ 조용한  — 멀리서 바라보는 차분한 모습
```

---

**프롬프트 변환 예시:**

```
입력값:
  품종=말티즈, 체형=소형, 나이=성견, 털색=#FFFFFF
  에너지=high
  성격키워드=[꼬리 세게 흔듦, 졸졸 따라다님, 이름에 바로 반응]
  무드=활발한

↓ AI 프롬프트 자동 생성:

"A small Maltese dog with pure white fluffy coat, tail wagging vigorously
at high speed, trotting eagerly toward owner with fully open bright eyes,
body leaning forward with excitement and high energy. Responsive and
social personality. Rim backlight, pure black background, hologram optimized."
```

### 3.4 선택 입력 — 기존 영상 (Phase 2)

| 항목 | 스펙 |
|------|------|
| 지원 포맷 | MP4, MOV |
| 최대 크기 | 100MB |
| 권장 길이 | 5초~3분 |
| 활용 | AI가 행동 패턴 자동 분석 → Step 2 성격 키워드 자동 채움 |
| 개인정보 처리 | 사람 얼굴 자동 감지 → 블러 처리 후 분석 |

## 4. 출력 데이터 스펙

### 4.1 기본 생성 영상 (웹 재생 + 다운로드)

| 항목 | 스펙 |
|------|------|
| **포맷** | **MP4 (H.264)** |
| **해상도** | **1920 × 1080 (Full HD)** |
| **프레임레이트** | **30fps** |
| **영상 길이** | 5~8초 (루프 재생 최적화) |
| **배경색** | 순수 검정 (#000000) — Pepper's Ghost 홀로그램 최적화 |
| **오디오** | 없음 (무음) |
| 예상 파일 크기 | 10~30MB |
| 다운로드 파일명 | `everpet_{반려동물이름}_{YYYYMMDD}.mp4` |

### 4.2 홀로그램 기기 최적화 버전 (자동 추가 생성)

| 항목 | 스펙 |
|------|------|
| 포맷 | MP4 (H.264) |
| 해상도 | 1080 × 1080 (정사각형, Pepper's Ghost 투영비율) |
| 프레임레이트 | 30fps |
| 특이사항 | 피사체 테두리 림라이트 강조 / 순수 검정 배경 |

### 4.3 썸네일 (자동 생성)

| 항목 | 스펙 |
|------|------|
| 포맷 | JPG |
| 해상도 | 400 × 400 |
| 생성 시점 | 영상 완료 시 자동 추출 (1초 지점) |
| 용도 | 보관함 그리드 표시 |

### 4.4 에러 처리

| 에러 상황 | 표시 문구 | 처리 |
|---------|---------|------|
| 포맷 불일치 | "JPG 또는 PNG 파일만 올릴 수 있어요" | 업로드 차단 |
| 파일 크기 초과 | "10MB 이하 사진을 올려주세요" | 업로드 차단 |
| 반려동물 미감지 | "사진에서 아이를 찾지 못했어요. 아이가 잘 보이는 사진으로 다시 올려주세요" | 재업로드 유도 |
| AI 생성 실패 | "잠깐 문제가 생겼어요. 한 번 더 시도해볼게요" | 자동 재시도 1회 |
| 재시도도 실패 | "오늘은 만들기가 어렵네요. 크레딧은 그대로 돌려드릴게요" | 크레딧 자동 반환 |

## 5. 유저 스토리

### 온보딩 & 프로필

**US-01** `MUST`
나는 **처음 방문한 보호자**로서, **웹 브라우저에서 앱 설치 없이 바로 시작**하고 싶다.

> ✓ 반응형 웹. PC·모바일 브라우저 모두 동작.
> ✓ 카카오 / 이메일 가입 30초 이내.

---

**US-02** `MUST`
나는 **보호자**로서, **우리 아이의 행동 특성을 직접 선택해서 알려주고** 싶다.
AI가 모르는 걸 내가 알려줘야 더 우리 아이다운 영상이 나온다.

> ✓ 에너지 레벨 1개 필수 선택.
> ✓ 성격 키워드 최대 3개 선택 (선택 안 해도 진행 가능).
> ✓ 처음 등록 시 1회 입력, 이후 생성마다 자동 적용.

---

### 영상 생성

**US-03** `MUST`
나는 **보호자**로서, **JPG 또는 PNG 사진 1장(최대 10MB)을 올리면 영상**이 만들어졌으면 한다.
2K 이상 선명한 사진이 더 좋다는 안내도 받고 싶다.

> ✓ JPG, PNG 지원 / 최대 10MB.
> ✓ 업로드 후 2초 내 미리보기 + 해상도 품질 피드백.

---

**US-04** `MUST`
나는 **보호자**로서, **오늘 기분에 맞게 무드를 골라** 영상을 만들고 싶다.
같은 아이라도 오늘은 활발하게, 내일은 조용하게 보고 싶을 때가 있다.

> ✓ 무드 3종 (포근한 / 활발한 / 조용한) 매 생성마다 선택.
> ✓ 같은 사진 + 다른 무드 = 다른 느낌의 영상.

---

**US-05** `MUST`
나는 **보호자**로서, **3분을 기다리는 동안 빈 화면이 아닌 무언가를 보고** 싶다.

> ✓ 대기 화면: 업로드한 아이 사진 + "콩이를 만들고 있어요" 문구.
> ✓ 3단계 진행 표시 + 예상 시간 카운트다운.
> ✓ 탭을 닫아도 완료 시 이메일 알림.

---

**US-06** `MUST`
나는 **보호자**로서, **완성된 영상을 1920×1080 MP4로 다운로드**하고 싶다.
내 컴퓨터나 TV에서도 보고 싶기 때문이다.

> ✓ 생성 완료 화면에 [다운로드] 버튼.
> ✓ 파일명: `everpet_콩이_20260322.mp4`
> ✓ Free/Basic은 워터마크 포함, Premium은 없음.

---

**US-07** `MUST`
나는 **보호자**로서, **보관함에서 언제든 이전에 만든 영상을 다시 볼** 수 있어야 한다.

> ✓ 보관함: 반려동물별 탭 + 날짜순.
> ✓ 썸네일 탭 → 전체화면 재생.

---

**US-08** `SHOULD`
나는 **보호자**로서, **기존에 찍어둔 우리 아이 영상을 올리면 AI가 특징을 파악**해줬으면 한다.

> ✓ Phase 2 기능. 영상 업로드 → 행동 분석 → 성격 키워드 자동 채움.

---

**US-09** `SHOULD`
나는 **기기를 가진 보호자**로서, **앱에서 선택하면 방 안 홀로그램 기기에서 바로** 나타나길 원한다.

> ✓ Phase 2. 같은 Wi-Fi 기반 연동.

## 6. 핵심 플로우

### 영상 생성 플로우 (Happy Path)

```
웹 접속 (PC or 모바일 브라우저)
  └→ 가입 / 로그인 (30초)

반려동물 등록 (최초 1회)
  ├ 이름 + 품종 + 체형
  ├ 털색 (자동 추출 or 선택)
  ├ 에너지 레벨 선택      ← 개체 맞춤 핵심
  └ 성격 키워드 선택 (최대 3개) ← 개체 맞춤 핵심

사진 업로드
  ├ JPG/PNG, 최대 10MB, 권장 2K
  └ 품질 피드백 즉시 표시

무드 선택 (매번)
  └ 포근한 / 활발한 / 조용한

[생성 시작]
  ├ 사진 준비 중   (~10초) — 배경 제거 + 림라이트
  ├ 아이를 만드는 중 (~2분) — AI 영상 생성
  └ 거의 다 됐어요 (~20초) — 인코딩

완료!
  ├ 전체화면 영상 자동 재생 (루프)
  ├ [다운로드]       MP4 1920×1080
  ├ [보관함에 저장]
  ├ [홀로그램으로 보기]  (Phase 2)
  └ [공유하기]       카카오 / 링크
```
